The Problem: Move Zeroes

My Solution:

Modified in-place

def moveZeroes(nums: List[int]) -> None:
    zero_count = 0
    
    while 0 in nums:
        nums.pop(nums.index(0))
        zero_count+=1

    zeroes = [0]*zero_count
    nums += zeroes
